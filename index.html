Git Erklärung

git status = zeigt den aktuellen status der Arbeit an (häufig benutzen) git log = zeigt Daten wie Datum, letzter user etc. (--oneline kurzform anzeige) git pull = läd die aktuelle version aus der cloud runter in das local repository git diff = zeigt veränderungen im code an git branch = erzeugt ein neues branch git branch -a = zeigt alle branches an die wir haben in das local repository git checkout name= wechselt auf beliebige branch git checkout -b = erstellt ein branch und wechselt in die branch git push -u origin "name" = erstellt neues repository in der remote cloud git merge "test" = überschreibt ein jeweiligen branch git branch -d = delete branch on your local repository git push origin --delete = delete branch on your remote git clone = macht eine kopie von der jeweiligen repository git checkout + ID = kann man zwischen deinen versionen hin und her switchen ver git commit --amend "name"= überschreibt den letzten commit git reset --soft HEAD~1 = löscht den letzten commit, behält den inhalt git reset --hard Head = löscht den letzten commit, löscht den inhalt
